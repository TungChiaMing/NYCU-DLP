## Usage
### Preprocessing

- Place the dataset to the D:
- `python ./database/preprocessing.py`

### Run training
```
python ./main.py
```

### Run validation to get Highest Accuracy
```
python ./main.py --mode demo
```  

### Run testing to get .csv for kaggle
```
python ./main.py --mode test
```
